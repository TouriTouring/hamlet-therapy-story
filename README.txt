Hamlet Does Therapy Story

Files:
- index.html
- images/
- sounds/

How to use:
1. Put index.html in the main Hamlet-Therapy-Game folder.
2. Put the art files in the images folder with these exact names:
   - office_neutral.png
   - office_concerned.png
   - office_surprised.png
   - office_scared.png
   - window_closed.png
   - window_open.png
   - clipboard_unbroken.png
   - clipboard_broken.png
   - ending_turningpoint.png
   - ending_delulu.png
   - ending_ontherun.png
3. Put the sound files in the sounds folder with these exact names:
   - bg_music.mp3
   - chair_screech.mp3
   - clipboard_break.mp3
   - window_open.mp3
4. Double-click index.html, or right-click it and open in a browser.

Notes:
- Music begins after clicking Start Story because browsers block autoplay until user interaction.
- This version is a simple branching visual-novel style HTML app.
